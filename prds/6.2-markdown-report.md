# PRD 6.2: Markdown Summary Generator

## Overview
Generate concise Markdown summary report for quick review.

## Deliverables
- `src/reporting/markdown-report.ts`

## Interface

```typescript
export interface MarkdownOptions {
  outputPath: string;
  maxFailedFiles?: number; // limit list, default 50
  includeConfig?: boolean;
}

export async function generateMarkdownReport(
  db: Database,
  runId: number,
  options: MarkdownOptions
): Promise<string>;
```
